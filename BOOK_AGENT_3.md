# Books_9 — Агент адаптации книг

## Что делает агент

Редактор загружает один файл с оригинальным текстом книги.
Агент автоматически создаёт все файлы нужные для ридера:
- Адаптированные версии по уровням (A1 / A2 / B1 / B2)
- Переводы на все языки
- Метаданные книги (book.json, описание)
- Картинки к главам (опционально)

Результат — готовый ZIP архив.
Или агент сам пушит в GitHub — редактор не касается репозитория.

---

## Как это работает (шаги)

```
1. Редактор загружает book.txt (оригинал на английском)
        ↓
2. Агент определяет уровень оригинала
   Анализирует первые 500 слов → "Это C1"
        ↓
3. Каскадная адаптация (только английский)
   C1 → B2 → B1 → A2 → A1
   Каждый шаг отдельный прогон — качество лучше чем прыжок сразу в A1
        ↓
4. Параллельный перевод (все уровни × все языки одновременно)
   каждый уровень → UK, RU, PL, DE, ES, FR, IT, PT, CZ
        ↓
5. Генерация картинок к главам (опционально)
   Claude анализирует каждую главу → prompt → GPT Image 1 Mini
        ↓
6. Генерация метаданных
   book.json, desc.en.txt, desc.uk.txt ... (все языки)
        ↓
7. Редактор проверяет результат прямо на странице
   Оригинал слева — адаптация/перевод справа, можно редактировать
        ↓
8. Скачать ZIP или автопуш в GitHub
```

## Поддерживаемые форматы загрузки

| Формат | Поддержка | Плюсы | Минусы |
|--------|-----------|-------|--------|
| **EPUB** ⭐ | ✅ | Главы, метаданные (название/автор) автоматически | — |
| **TXT** | ✅ | Просто | Нет структуры, название вводить вручную |
| DOCX | 🔜 позже | Знакомо редакторам, заголовки глав | Нужна доп. библиотека |
| PDF | ❌ | — | Нестабильное качество извлечения текста |

**EPUB — рекомендуемый формат.** Внутри уже есть структура глав, название, автор, язык оригинала — редактор не вводит это вручную, агент читает сам.

**TXT — запасной вариант** если EPUB нет. Главы агент определит по двойному отступу или `[квадратным скобкам]` — так же как ридер.

---

## Что редактор делает

1. Открывает страницу агента в браузере
2. Загружает файл книги (EPUB или TXT)
3. Проверяет название и автора (при EPUB заполняется автоматически)
4. Загружает обложку (картинка)
5. Выбирает языки (галочки) — можно не все 9
6. Выбирает стиль картинок: акварель / минимализм / комикс (если нужны)
7. Нажимает **Запустить**
8. Ждёт 5-10 минут (видит прогресс по шагам)
9. Проверяет результат на странице
10. Нажимает **Скачать ZIP** или **Опубликовать** (автопуш в GitHub)

**Никаких технических решений от редактора не требуется.**

```

      chapter_1.jpg      ← картинка к главе (опционально)
    b2/
      book.en.txt
      book.uk.txt ...
    b1/ ...
    a2/ ...
    a1/ ...
```

### Нужен ли Supabase?

| Этап | Supabase нужен? |

### Домены — что платно, что бесплатно

| Что | Цена |
|-----|------|
| `yourproject.pages.dev` (домен от Cloudflare) | **Бесплатно** |
| Купить свой домен (`yourdomain.com`) | ~$10-15/год |
| Подключить свой домен к Cloudflare Pages | **Бесплатно** |
| Поддомены (`editor.yourdomain.com`) | **Бесплатно** если домен уже в Cloudflare |

**Если домен уже есть:** переносишь DNS в Cloudflare → все поддомены бесплатно.
**Если домена нет:** можно начать на `.pages.dev` бесплатно, потом купить домен.

    ↓ (авторизован через Supabase Auth)

## Технический стек агента

- **Фронтенд:** HTML + JS (Cloudflare Pages)
- **Backend:** Cloudflare Worker (API ключи, проксирование)
- **Определение уровня + адаптация:** GPT-4o API
- **Перевод:** GPT-4o-mini API
- **Картинки:** GPT Image 1 Mini API
- **ZIP:** JSZip (в браузере, без сервера)
- **Автопуш:** GitHub API (опционально)
- **Авторизация редакторов:** Supabase Auth
